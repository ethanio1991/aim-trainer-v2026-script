## What It Is

Aim Trainer is a browser-first practice game built to sharpen aim fundamentals through brief, repeatable drills. Because it runs as an HTML5 experience, it is easy to launch in the browser and return to whenever you need a quick warm-up before playing or training.

The focus is on skills that matter in real aim practice: target acquisition, mouse movement control, reaction timing, and click consistency. Since it uses vanilla JavaScript and can run offline, it stays lightweight and straightforward for players who want a no-frills training front end without extra setup.
